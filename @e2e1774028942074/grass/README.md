# All About Grass - Website

A beautiful, informative website about grass and its importance in our ecosystem.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth animations and hover effects
- **Educational Content**: Comprehensive information about grass types, benefits, and care
- **Modern Styling**: Clean, professional design with consistent color scheme

## Structure

- `index.html` - Main website content
- `style.css` - Complete styling with CSS variables and responsive design
- `script.js` - Interactive features and animations

## Content Sections

1. **Hero Section** - Eye-catching introduction
2. **Introduction** - What is grass?
3. **Types of Grass** - Cool-season vs warm-season grasses
4. **Benefits** - Why grass matters to our ecosystem
5. **Care Tips** - How to maintain healthy grass

- HTML5
- CSS3 with CSS Variables
